# Lekhajokha (लेखाजोखा) - Governance Reform Tracker

Lekhajokha is an independent, real-time, and data-driven platform designed to track the Nepal Government's 100-Point Action Plan (शासकीय सुधार कार्यसूची) announced on March 28, 2026. This platform strictly monitors the progress of government commitments, ensuring civic accountability and transparency.

## Features

- ⏱ **100-Day Live Tracking HUD**: Interactive 3D countdown timer with progress radar.
- 📊 **Detailed Data Model**: Accurately tracks all 100 verified commitments, including mega-points with multiple discrete deadlines and sub-commitments.
- 🌗 **Premium Theming**: Support for both immersive Dark Mode and clean Light Mode.
- 🌐 **Bilingual Support**: Full toggleable interface between English and Nepali.
- 📱 **Fully Responsive**: Architected using Next.js and Tailwind CSS for mobile, tablet, and desktop perfection.
- 🕵️ **Data Integrity**: Synced perfectly with the official publication, breaking down complex promises into trackable KPIs.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4)
- **UI Components & Animations:** Framer Motion, Lucide React
- **State Management:** Zustand
- **Architecture:** Client/Server Component hybrid

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/DiwanMalla/lekhajokha.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Methodology

This platform independenty verifies commitments based purely on the exact document approved at the first Cabinet meeting of Prime Minister Balendra "Balen" Shah's government (Chaitra 13, 2082). We do not editorialize commitments—we track the literal letter of the action plan.
